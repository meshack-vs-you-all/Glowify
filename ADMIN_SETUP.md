#### Hero Section

   - Add subtitle (e.g., "New Collection")
   - Add title (e.g., "Discover Our Latest")
   - Add description text
   - Add button label (e.g., "Shop Now")

#### Featured Collection Section
1. Click on "Featured Collection" section
2. Select a collection from dropdown
3. Set "Products to show" (default: 8)
4. Add heading (e.g., "Featured Products")

#### Trust Badges Section
1. Click on "Trust Badges" section
2. Click "Add block" for each badge
3. For each badge:
   - Upload icon (optional, or use default)
   - Add title (e.g., "Free Shipping")
   - Add text (e.g., "On orders over $50")

#### Newsletter Section
1. Click on "Newsletter" section
2. Add heading (e.g., "Stay in the know")
3. Add description text
4. Customize button label if needed
